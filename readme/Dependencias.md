# Dependências do Projeto

Este projeto utiliza várias bibliotecas e pacotes para funcionar corretamente. Aqui está uma lista das principais dependências e suas funções:

- **express**: Framework web rápido e minimalista para Node.js.
- **body-parser**: Middleware para analisar corpos de requisição.
- **bcryptjs**: Biblioteca para hash de senhas.
- **jsonwebtoken**: Implementação de JSON Web Tokens (JWT) para autenticação.
- **dotenv**: Carregar variáveis de ambiente a partir de um arquivo `.env`.
- **swagger-ui-express**: Middleware para servir a documentação da API gerada pelo Swagger.
- **swagger-jsdoc**: Biblioteca para gerar a documentação Swagger a partir de comentários no código.
- **joi**: Biblioteca de validação de dados para JavaScript.
- **fs**: Biblioteca do Node.js para manipulação de arquivos.

Para instalar todas as dependências, execute:
```bash
npm install express body-parser bcryptjs jsonwebtoken dotenv swagger-ui-express swagger-jsdoc joi
